# WorkerJS_CloudFlare_ImageBed

基于JS+HTML的图床项目开发框架，包括前端（pages，html）和后端（worker，js）等。

以及测试用py上传工具和构建、脚本。

仅供学习Worker开发使用，违规使用后果自负。

大部分接口不一定可用，仅供教学学习测试验证。
